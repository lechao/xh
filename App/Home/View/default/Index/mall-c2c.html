ShopNC C2C电商系统介绍

多用户聚合型商城平台，适合C2C经营

ShopNC C2C电商系统基于PHP+Mysql技术的MVC架构开发的Web电子商务系统，它是以消费者对消费者（C2C）经营模式的电商系统，直接为消费者提供电子商务活动的网站平台，主要功能有：采用了缓存更新处理机制；首页和频道页有广告功能；支持团购、限时折扣、满减及优惠劵等促销模式；支付手段支持预存款支付、支付宝支付、财付通即时到账、财付通中介担保、银行汇款、网银支付、货到付款等多种支付手段；CMS（新闻发布）；广告管理；友情链接；交易邮箱提醒；交易信用评价；站内短信通知；平台分权限管理等功能；旨在为平台运营方提供一个功能完善强大的电子商务平台。
ShopNC C2C电商系统具备跨平台特性，可运行于Linux/FreeBSD/Unix及微软Windows 2000/2003/2008/XP/NT等各种操作系统环境下。推荐使用Linux/FreeBSD操作系统。
- 主流WEB服务器（如Apache、Nginx、IIS等）
- PHP 5.4+
- MySQL 5.5+

ShopNC C2C电商系统的功能特点

从商城功能模块读解其用途与特征，以及功能亮点。

  • 在线支付管理

    支持支付宝、财付通、预存款、银行汇款、网银支付、移动支付等方式。

  • 多级分类

    商品可实现多级分类，从而满足网站多元化商品信息的需求。

  • 商品搜索

    多种搜索方式；如：商品、团购、店铺、资讯搜索等多种搜索方式。

  • 信用评价

    拥有信用评价功能、可对交易双方作出信用评价，并自动统计用户的信用指数。

  • 店铺类型

    根据店铺类型、等级可设置不同的权限；如：商品发布数量、店铺功能区分。

  • 店铺模板

    系统预置5套精美模板供店铺选择使用，切换方便。

  • 主营品牌设置

    选择品牌和店铺类型(如：经销、专卖、专营、旗舰)。

  • 商品信息发布

    发布商品介绍、图片、价格等商品信息。

  • 商品批量导入

    可批量导入淘宝商品图片和数据。

  • SEO增强

    支持商品展示页面关键信息的独立设置，更易于搜索引擎收录。

  • 水印管理功能

    支持支持图片和文字水印方式。

  • HTML编辑

    信息的添加和更新采用HTML编辑器，实现所见即所得。

  • 站内短信管理

    接收和回复站内短信站管理功能。

  • 广告系统

    支持支持图片和文字水印方式。

  • 数据库管理

    可以后台进行数据库备份还原管理。
