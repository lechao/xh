商城频道 聚焦商品

当更多的商家开店、更多的商品上架，每件商品都想成为“首页明星”的时候，该如何解决？该系统给您一个满意的答案。商城频道功能绑定商品分类，可视化编辑，自定义导航、楼层模块，聚焦更多商品。

模块化理念构思，通过对顶部与楼层布局模板的选择组合，绑定商城商品分类，设计构成商城频道页面。

顶部楼层模块可自定义导航菜单、菜单图标；可视化焦点幻灯大图与三联轮播广告自由添加，丰富多变。

楼层模块选择预设布局，可视化内容设计，添加多组手动选择推荐商品或自动排序规则显示某分类商品内容。

商家促销

商城大促、季节打折、品牌推广这些活动都离不开对应专题页面，为此，系统增加了专题页编辑制作功能；配合主导航链接设定以及首页、频道页链接操作选择，用户可以从任何地方跳转到精彩的专题页面中。

专题页面的背景提供更多选项，不但可以选择颜色、上传图片，更可以对背景填充方式、高度等逐一进行设置。

灵活使用图片热点功能，可以将一副设计好的图片进行上传、并添加热点区域以及操作方式，组合出多变的页面。

单图链接、多图轮换、自定义内容编辑、商品推荐；丰富的编辑功能让专题页面的制作轻松简单。

预售定金 赠品套装

商品促销是商城系统销售过程中最重要的营销手段，增设店铺商品全款预售、定金预售、优惠套装、商品赠品等多种必备实用促销功能。

定金预售

  • 定金预售商品可选择添加商品SPU，并可勾选参加活动的商品SKU以及定金比例、预售时间段等参数选项。
  • 定金预售商品在预售期间客户首付定金购买，并在规定的期限内付清尾款，商家提供发货。
  • 定金预售不支持批发模式商品，添加后不能随意修改定金比例及时间段设定。

优惠套装

  • 选择添加最多五个商品，并可独立设定所添加商品SKU以及套装内部折扣价格。
  • 商品详情页面可展示多个优惠套装切换，用户下单可选择商品SKU规格。
  • 购物车内套装以组的形式展现，可整组添加数量并可合并加入收藏等等。

商品赠品

  • 开启促销活动后选择添加活动商品；确定活动商品后进行赠品捆绑，最多可选择添加五种赠品，赠品可选择商品SKU值。
  • 赠品活动仅支持零售型商品，赠品价格不计算为订单价格，售出赠品将减少商品库存存。
  • 移动端对应更新赠品显示。

微信营销 多人拼团

新增微信管理功能，平台可设置公众帐号并对公众帐号菜单、回复等常用功能进行设置，逐步强化电商品台与微信之间的深度融合。

商家发起拼团活动制定多人拼团规则、人数、时间段以及参与活动的商品SKU。

用户在微信wap商城购买活动商品后系统自动开团，根据规则判断拼团成功与否。

已购会员分享拼团商品地址给微信好友，邀请好友加入拼团，人数到达后拼团活动成功。

更多功能

  1. 1新增商家中心店铺装修功能，可视化编辑店铺首页、专题页面，可设置背景项、单图链接、热点图片、自定义内容、推荐商品等。
  2. 2新增用户投诉功能，当买卖双方发生交易问题无法自行解决的时候，会员可从已有的订单、退款单申请投诉，由平台进行仲裁解决，保证利益。
  3. 3新增店铺搜索功能，可在顶部主搜索条点击切换至店铺项，输入店铺名称关键字查找，以列表形式展现店铺基本信息、最新商品、动态评分等。
  4. 4新增商品搜索多选属性索引功能，将原有的属性选择只能使用单一属性值变为可多选形式，提高商品索引选择范围。
  5. 5增加商城前台会员侧边栏功能、可显示会员信息、红包优惠券、购物车、收藏、浏览记录等多项常用信息、并可激活在线IM聊天。
  6. 6在线客服功能在原基础上进行改进与完善，增加聊天记录查询、可本地上传图片以及聊天窗口自动将商品链接转化成介绍内容等。
  7. 7移动端更新功能：店铺搜索、到货通知、商品列表页面多属性索引、单品赠品、预售定金、IM商品链接地址转换与PC同步，优化界面、用户体验。
  8. 8商城系统后台增加商品分类手动推荐商品设置功能。
  9. 9商家中心相册目录建立、图片移动、删除等功能。
  10. 10商城会员部分增加商品到货通知功能，手机短信、站内消息等。
  11. 11商城系统优化缓存体系，进一步提高整体性能。
  12. 12商城系统优化队列机制，进一步提高处理速度。
  13. 13商城系统支持使用阿里云对象存储(OSS)替代本地存储。
  14. 14商城系统图片服务支持多域名部署，提高图片并行加载速度。
